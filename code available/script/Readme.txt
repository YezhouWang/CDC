NOTE: you need to install BrainSpace Toolbox before run these script.

bulid_FDC: Construct a functional distance coefficient map

bulid_mpc: Construct a microstructure profile covariance matrix

estimate_gradients: estimate gradients from multi-scale data, e.g. neuroimaging or histological data

transcriptomic_analysis: a series of analyses on gene expression data, e.g. gene enrichment analysis, cell-type specific analysis and gene co-expression analysis.
